About

I am a physicist engineer with experience in wave-based measurement systems, signal analysis and scientific computing. My work has focused on the evaluation of real ultrasonic inspection data, the interpretation of signal quality in operational environments and the development of custom tools to support technical analysis and process improvement.

I use programming not only for scientific analysis, but also for building practical engineering applications in Python and C++. For me, software is a way to understand physical systems, automate technical workflows and make complex analysis more accessible and useful in real engineering contexts.

Projects

Ultrasound Imaging: From Signal to Image

Python/PyQt-based technical application for generating, visualizing and processing ultrasound-like signals from point reflectors.

  • RF A-scan generation from pulse models and reflector parameters
  • Interactive visualization of time-domain signals and depth-related echoes
  • Processing workflow toward B-scan representation and imaging-related metrics
Repository Project notes

Wave Propagation Simulation

Exploratory simulation environment to visualize how a wave propagates, interacts with simplified reflectors and returns to a sensor.

  • Conceptual model of source, medium, reflectors and receiver
  • Visualization of wave travel, reflection and return paths
  • Future direction: finite-difference simulation and parameter-controlled geometries
